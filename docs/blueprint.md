# **App Name**: GitDo

## Core Features:

- Add Task: Allows users to input and add new tasks to the todo list.
- Mark Complete: Enables users to mark tasks as complete, visually indicating their status.
- Delete Task: Provides a way for users to remove tasks from the list with a confirmation prompt.
- Data Persistence: Saves the task list in the browser's localStorage to persist data across sessions.
- AI Task Suggester: Suggest tasks using an AI tool that takes into consideration time of day and previous tasks to intelligently manage schedule.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) for a calming and focused ambiance.
- Background color: Lightest blue (#F0F8FF), almost white, maintaining a clean and unobtrusive look.
- Accent color: Pale violet (#BDB2FF) to draw attention to interactive elements.
- Font: 'Inter', a grotesque-style sans-serif, is recommended for both headlines and body text.
- Simple line icons for task actions (add, delete, complete).
- Clean, single-column layout optimized for readability and ease of use.
- Subtle animations for task transitions (completion, deletion) to provide visual feedback.
